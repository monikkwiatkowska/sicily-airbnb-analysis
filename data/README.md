# Data

## Source
Download Sicily Airbnb data from [Inside Airbnb](http://insideairbnb.com/get-the-data.html)

## Files Not Included
- `listings.csv` - Raw data (too large for GitHub)
- `listings_cleaned.csv` - Processed data (too large for GitHub)

## Instructions
1. Visit Inside Airbnb website
2. Download Sicily/Palermo listings data
3. Place in this directory to run the analysis
